「移動…」メニュー

「検索」メニューの「移動…」（Cmd + L）で移動シートが表示され、移動先の範囲指定が可能です。

移動パネルでは「行」「文字」プルダウンメニューを選択し、選択したい数値を入力して「選択」ボタンをクリックします。

  • 「行」を選択して「3」を入力
    => 3行目全体が選択される
  • 「行」を選択して「3:2」と入力
    => （「3行目からの2行を選択」という意味になり）3・4行目が選択される
  • 「行」を選択して「3:-1」を入力
    => （「3行目から、終わりから1行目を残して選択」という意味になり）全部で10行ならば、3〜9行目まで選択される

「文字」が選択されている時も、同様に意味付けられ、選択移動します。
