Edit preferences

The Edit preferences let you set the behavior of text view while text editing and its display.

Behavior
Enable smart insert and delete
When this option turned on, when you paste or drag a word in between other words, a halfwidth space will be automatically added between the words to maintain that boundary. Meanwhile, any extra spaces will be automatically removed when you cut/delete a word. (This function only has meaning when you edit in English.)
Enable smart quotes and dashes
When this option turned on, inserted quote (") will be automatically substituted with matched quote (“ or ”). With dash, it behaves also the same.
Drag selected text immediately
In order to drag selected text, you need to hold the mouse button pressed more than 0.3 seconds before starting dragging, but with this option turned on, you can reduce the waiting time and start dragging immediately. (This is an experimental implementation.)
Swap “¥” and “\” keys
When this option turned on, you can input a backslash (\) using the key binding for the Yen sign (¥), and vice versa.
Highlight matching braces “( )” “[ ]” “{ }”
When this option turned on, CotEditor briefly highlights the corresponding opening brace when you write a closing brace.
Highlight "< >"
If this is on, highlighting will apply to angle brackets “< >” as well.
Indentation
You can set the indentation behavior in the text view.
Auto indent
When this option is turned on, CotEditor automatically indents each line you create by hitting the Return/Enter key. The new line will begin at the same indentation point as the previous line.
Expand tabs to spaces
When this option turned on, CotEditor automatically converts tab characters to spaces as you type.
Indent width:
You can specify the number of spaces to which a tab character is expanded.
Comment
You can set the comment-out/uncomment behavior in the text view.
Append a space to comment delimiter
When this option turned on, a single space character will be appended between comment delimiters and commented out strings.
Always from line head
When this option turned on, the selected lines will be commented out always from line head.
Line wrapping
You can set the line wrapping behavior in the text view.
Soft wrap lines
When this option is turned on, CotEditor soft-wraps long lines so that you can view them within the window width. Otherwise, CotEditor shows horizontal scroll bar without wrapping text.
Show Invisibles
You can transform invisible characters — space, tab, line ending, control character — into alternative symbols in order to make them viewable. Select the symbol you wish to use from the drop-down menu.
The default font you set in the Appearance pane applies to these symbols as well.
Completion
You can set the completion behavior in the text view.
Auto complete
When this option is turned on, completion list will be automatically shown if words to complete exist.
Completion list includes words in:
Here, you can change the range of possible completions that appear in the completion list.
Document only
The list shows only the words appear in your document.
Syntax Style only
The list shows only the words defined by your syntax style (in the case you've not specified any completions, the words generated by the syntax coloring rule appear).
Document + Syntax Style
In addition to the in-document words, the words defined by your syntax style appear in the list.
Document + Syntax Style + Standard Words
In addition to all of the above, the list shows standard words as well.
