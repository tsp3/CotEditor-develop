Regular Expression Syntax

CotEditor's syntax coloring and outline menu functions use ICU regular expression and the find panel uses Onigumo (Oniguruma-mod) regular expression engine. Please take notice that their regular expression syntax are slightly different from each other. See the following references to make sure they work properly.
